Description UW Facilities manages shipping containers in the corporation yard. Each maintenance shop typically owns one or more containers in which they can store their equipment. For safety purposes there is a need to track the point of contact for each container. The feature class contains the location of all containers in corp yard 1, 2 and 3 along with the point of contact.
