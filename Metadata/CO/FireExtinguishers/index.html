Description This feature class contains the location of all installed fire extinguishers that are maintained by Campus Engineering & Operations on the Seattle Campus of the University of Washington. These extinguishers are located on a room by room basis for each building. Additional attributes include the type, size and inspection sequence. This feature class serves two purposes, for CEO they are able to record when inspections were performed on each extinguisher to ensure compliance and EHS uses this data to produce evacuation map drawings for each floor in a building.
