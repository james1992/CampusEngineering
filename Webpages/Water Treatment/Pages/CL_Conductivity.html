Cooling Tower Conductivity Troubleshooting



    Below Range

  • Check the Conductivity Graph.
  • If the value is less than 600, try to blowdown the tower.
  • If it blowsdown, contact NALCO.
  • If it does not blowdown, check for overflow or leak.

    Above Range

  • Check the Conductivity Graph.
  • If the value is greater than 1200, try to blowdown the tower.
  • If it blowsdown, check for blocked solenoid.
  • If it does not blowdown, contact NALCO.
