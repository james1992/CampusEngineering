Cooling Tower Conductivity Troubleshooting



For more information, please contact NALCO.


    Below Range

  • Check the Conductivity Graph for trend.
  • Check Calibration against Hand Held meter
  • Has there been a planned condenser water loss (i.e. tower draining/cleaning)?
    • Yes - Conductivity is fine. Continue to monitor to make sure it increases.
    • No - Check for tower overflow, condenser leaks, excessive filter backwash or blowdown valve stuck open.

    Above Range

  • Check the Conductivity Graph.
  • Check calibration against hand held meter. If within 10%, go to next step. If >10% difference, call NALCO to calibrate.
  • If the water conductivity is greater than 1200 uS/cm, try to blow down the tower.
  • If it does not blow down: check solenoid, y-strainer, or replace solenoid.
  • If it does blow down, contact NALCO.
