Cooling Tower TRASAR Troubleshooting



    Below Range

  • Check the Inhibitor Graph.
  • If the value is less than 100, check to see if the cooling tower is trying to feed.
  • If it feeds check 3DT222 Inventory or Pump.
  • If it does not feed, contact NALCO.

    Above Range

  • Check the Inhibitor Graph.
  • If the value is greater than 200, check and see if it is trying to feed.
  • If the inhibitor is not trying to feed, no other action is needed.

    For more information, please contact NALCO.
