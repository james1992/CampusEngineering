How to use this site

To drop off an updated Master Excel file from OASIS use the dropdown menu labeled Menu and click on Excel Drop Off. This will open a new window in the same SharePoint folder as the previous GIS Dashboard. Drop off the Excel as you have in the past.

There is also a link that will take you back to the UW Facilities homepage.

In the top right corner of every webpage, you will find a link labelled UW Facilities GIS Home this link will take you back to the dashboard homepage.
